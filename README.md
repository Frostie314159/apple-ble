# Experimental library for interfacing with apple devices over BLE
## Warning
The library is not nearly finished. Expect breaking changes frequently.
## Supported BLE messages
* [x] AirDrop
* [x] AirPlay
    * [x] Source
    * [x] Target
* [x] AirPrint
* [x] FindMy
* [ ] HandOff
* [ ] HeySiri
* [ ] Homekit
* [ ] Magic switch
* [ ] Nearby
    * [ ] Action
    * [ ] Info
* [ ] Proximity pairing
* [ ] Tethering
    * [ ] Source
    * [ ] Target
